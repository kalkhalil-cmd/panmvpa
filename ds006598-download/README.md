This dataset contains the Precision targeting of Association Networks (PAN) data. Data were collected in 10 individuals over 6-9 separate two-hour scanning sessions. In all sessions, resting-state and task data was collected, totaling approximately 2 hours of resting-state and 6 hours of task-state data per individual. In the first two sessions, T1 data was also collected.

This data was processed using fMRIPrep 23.2.0, and the MNI-space derivatives as well as AFNI-compatible task timing files are provided here.

Details of this dataset are more fully described here - Ladwig et al., Precision fMRI reveals densely interdigitated network patches with conserved motifs in the lateral prefrontal cortex - https://www.biorxiv.org/content/10.1101/2025.07.24.666468v1.

Processing and analysis code is available here - https://www.github.com/GrattonLab/Ladwig-LPFC.
